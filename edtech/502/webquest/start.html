Web Quest:

A WebQuest is an inquiry-oriented lesson format in which most or all the information that learners work with comes from the web. These can be created using various programs, including a simple word processing document that includes links to websites.

Description:

This WebQuest will give the basic information on how to create a webpage. A webpage can be used to display information on any topic. It has information about the different tags that can be used that can be used to make webpage or website.

Student Instructions:

Work through the following 5 sections to complete the WebQuest
  • Introduction: Overview of the WebQuest
  • Task: The task to complete
  • Process: Step to follow to complete the task
  • Evaluation: How your work will be assessed
  • Conclusion: Final thoughts and future directions
